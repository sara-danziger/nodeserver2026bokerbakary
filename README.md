
# Node.js Express API for E-commerce

This project is a simple RESTful API built with Node.js and Express, designed to manage the entities `Order`, `User`, and `Product`. It provides CRUD operations to interact with these entities.

## Technologies Used

- Node.js
- Express.js
- MongoDB (via Mongoose ORM)
- Body-parser (for handling request bodies)
- dotenv (for environment variables)

## Project Setup

### Prerequisites

- [Node.js](https://nodejs.org/) v14 or later
- [MongoDB](https://www.mongodb.com/) (can be set up locally or use a cloud solution like MongoDB Atlas)
- Postman or any HTTP client for testing the API
